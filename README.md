오픈소스기초설계
----------------
###### 20233072 김경서

문제 정의
----------
*우천 확률이 낮아 경기장에 도착하였는데 비가 쏟아져 경기가 취소되는 경험을 함*

프로젝트 주제
-------------
> 잠실 야구장 우천 가능성 예측
>  > 목표 : 우천으로 인한 경제적 손해 최소화

해결 방안
-----------
* 신뢰할 수 있는 날씨 데이터 소스 선정
  * '초단기 강수 예측' 데이터 활용
    * 당일 경기 6시간 전 우천 취소 가능성 노출
